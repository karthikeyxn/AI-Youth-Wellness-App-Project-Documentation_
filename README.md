# AI Youth Wellness App

## AIM
To design an AI-powered Youth Wellness App for mood tracking, AI-based sentiment analysis, wellness recommendations, journaling, and chatbot support.

## PROCEDURE
- User authentication
- Mood tracking (rating + notes)
- AI sentiment analysis
- Wellness suggestions & chatbot
- Dashboard & reporting

## ALGORITHM
1. User registers/logs in
2. Enter mood rating + notes
3. Analyze sentiment
4. Provide suggestions
5. Update dashboard
